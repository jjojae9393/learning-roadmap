## Redis

Redis는 메모리 기반의 초고속 **키-값 저장소**입니다.
주로 캐싱, 세션 저장, 큐, 실시간 순위표 등에 사용됩니다.

### 왜 빠른가?

데이터를 디스크가 아닌 **메모리(RAM)** 에 두기 때문에 읽기/쓰기가 수 마이크로초 수준입니다.

### 대표 사용 예 — 캐싱

```text
1. 요청 도착
2. Redis에 캐시가 있나? → 있으면 바로 반환 (DB 안 거침)
3. 없으면 DB 조회 → 결과를 Redis에 저장(TTL 설정) → 반환
```

### 자주 쓰는 자료구조

- **String** — 단순 값/카운터
- **Hash** — 객체 형태
- **List** — 큐/스택
- **Set / Sorted Set** — 중복 제거, 순위표
