## HTML

HTML(HyperText Markup Language)은 웹 페이지의 **구조와 의미**를 정의하는 마크업 언어입니다.
프론트엔드의 가장 기본이 되는 뼈대입니다.

### 핵심 학습 포인트

- **시맨틱 태그**: `<header>`, `<nav>`, `<main>`, `<article>`, `<footer>` 등 의미에 맞는 태그 사용
- **폼과 검증**: `<form>`, `<input>`, `required`/`pattern` 등 기본 검증
- **접근성(a11y)**: 스크린리더, `alt`, ARIA 속성 고려

### 간단 예시

```html
<article>
  <h1>제목</h1>
  <p>본문 내용입니다.</p>
  <button type="button">더 보기</button>
</article>
```

> 아래 하위 주제에서 시맨틱 HTML·폼·접근성을 더 살펴보세요.
